# MQGT-SCF Closed Core Paper

This directory contains the canonical LaTeX source for the closed-core Theory of Everything paper.

## Compilation

```bash
make
```

Requires `latexmk` (install via `brew install latexmk` or TeX Live).

## Structure

- `main.tex` — Main paper source
- `response_to_comments.tex` — Addendum addressing review comments
- `refs.bib` — Bibliography (BibTeX format)
- `figures/` — Figure files (if any)

## Status

This is the reviewer-proof core physics paper. For the companion interpretation paper with teleological framing, axiarchy, and participatory ethics, see the companion paper directory (to be created).
